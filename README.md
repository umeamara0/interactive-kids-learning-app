Interactive Kids Learning App

A simple and engaging web application designed to help children learn alphabets through sound and visuals. This project demonstrates the use of JavaScript event listeners and audio integration.

 Key Features

* **Interactive Buttons**: Four dedicated buttons for alphabets (A, B, C, D).
* **Audio Feedback**: Clicking on an alphabet triggers a clear voice/sound (e.g., clicking 'A' says "Apple").
* **Kid-Friendly UI**: Vibrant colors and large, easy-to-click buttons for better accessibility.

 Tech Stack

* **HTML5**: For the application structure.
* **CSS3**: For colorful styling, button effects, and layout.
* **JavaScript (ES6)**: To handle click events and trigger audio playback.
